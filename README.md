# What is spooky-story?

This is a spooky story-mode game with roughly 30-40 different screens. Has jumpscares so be careful :)

Here is the [video](https://www.youtube.com/watch?v=1oJBgkQVEHE) showcasing the abilities of this wonderful website.

# Self Hosting Guide

### Prerequisites

- Node.js 20 or later
- npm or bun package manager

### Installation

1. Clone the repository:

```bash
git clone https://github.com/avalynndev/spooky-story.git
cd spooky-story
```

2. Install dependencies:

```bash
npm install
```

3. Run the development server:

```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

### Building for Production

```bash
npm run build
npm start
```

By following these steps, you can host the SpookyStory web app on your own server and make it accessible to users.
